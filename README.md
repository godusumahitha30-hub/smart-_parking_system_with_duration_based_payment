# 🚗 Smart Parking System

This project detects vehicles, tracks entry and exit, calculates parking duration, and generates billing with a live dashboard.

---

## 🔥 Features
- Vehicle Detection (YOLOv8)
- Tracking (DeepSORT)
- Parking Slot Monitoring
- Entry & Exit Time Tracking
- Automatic Billing System
- Live Revenue Dashboard
- Demo Video Mode (runs without upload)

---

## 🛠️ Tech Stack
- Python
- OpenCV
- YOLOv8 (Ultralytics)
- DeepSORT
- Streamlit
- Pandas

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
